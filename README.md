기타(느낀점)
남근형
기간이 짧게 주어지더라도 순서만 갖추어진다면 완성도 높은 결과물을 만들어 낼 수있다는 것을 알게되었다.  다양한 기술들을 활용하여 하드코딩 및 디버깅 시간을 줄일수 있었고, 덕분에 어떻게 개발할지 보다 무엇을 개발할지에 대해 집중할 수 있었다.
유시진
